# Study Note Dashboard V3 Stable 3.2.0

Bản này tiếp tục từ nhánh ổn định trước đó và giữ nguyên `manifest.key` + storage key để hạn chế mất dữ liệu khi cập nhật.

## Điểm mới
- Giữ nguyên dữ liệu cũ khi cập nhật đúng cách
- Dashboard mượt hơn, có `Today Focus`
- Full note view riêng
- Toolbar notes rõ hơn: sort + compact mode
- Thêm task input ngay trong dashboard
- Settings có thống kê dữ liệu và export từng backup
- Popup có stats nhanh

## Cập nhật không mất dữ liệu
1. Giải nén bản mới
2. Chép đè file mới vào đúng thư mục extension đang dùng
3. Mở `chrome://extensions` hoặc `edge://extensions`
4. Bấm **Reload**

Không nên bấm **Remove** nếu bạn muốn giữ dữ liệu cũ.

## Cài mới
1. Giải nén file zip
2. Mở `chrome://extensions` hoặc `edge://extensions`
3. Bật **Developer mode**
4. Chọn **Load unpacked**
5. Chọn thư mục này
